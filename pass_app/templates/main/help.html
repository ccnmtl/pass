What browsers should I use for this site

PASS is best viewed in the following browsers:

  • Google Chrome
  • Mozilla Firefox
  • Apple Safari
  • Internet Explore for Windows versions 8 and above.

What is a UNI login?

A UNI is the username and password assigned to you if you are a student, faculty or staff member of Columbia University. Please visit the Columbia Univesity Information Technology (CUIT) website for more information.

How do I create a new account?

  • Click "Need a new account?" link on the login page.
  • Once you've selected your username and password, you will receive an email. Please check it and click on the confirmation link to activate your PASS account.
  • Log in after you've activated your account using the username and password you set at registration to access PASS modules.

I forgot my account username/password. What should I do?

If you're logging in using your UNI, please visit the CUIT website to reset your password.

If you're logging in using your PASS account, please send us an email and we will help you reset your account.

The interactive activities are not working.

First, check if your browser is listed on the supported browser list of this website. Then check the JavaScript setting for your browser. If it is disabled, turn JavaScript on.

If the activities are still not working properly, please send us an email detailing what went wrong, which operating system you're on, and the browser version you're using.

Can I undo my responses in the activities?

Unfortunately, you cannot undo the responses within the activities that you've submitted. However, please send us an email to discuss a reset to the activities.

I entered incorrect information on the Feedback/Register page. Can I change it?

Please send us an email to request help on reseting your registration and feedback.

The videos are not playing on my browser.

This site uses Flash video, and it requires the latest version of Adobe Flash Player. Please install the player and try again.

If the video playback problem still persists, please send us an email detailing what went wrong, which operating system you're on, and the browser version you're using.

Can I jump ahead to another section, or go back?

PASS modules are serial learning environment, where you can move forward a section at a time. You can return to visited sections using the drop-down menu bearing the name of the module.
